# workBlog
项目博客
---

## 介绍
一个前后端分离的博客项目
技术栈：
- springboot
- vue
- redis
- mybatisplus

## 具体实现：
基于jwt的登录注册功能
基于redis的缓存功能
基于localthread的本地线程保存用户信息功能
